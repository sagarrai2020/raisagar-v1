My workflow

Over the years i've built many websites in many different topics. I've experimented many different techonologies to make website look unique, visually appealing, bug free and how to make website load fast and SEO friendly and how to manage a website for future updates and customizations. based on my personal experience and years of experiments, now everything i built from scratch (no CMS platforms such as Wordpress). Here is the details of how i build my websites from scratch.

  • Gathering Information

    First of all i gather relavant information from the client so that i can identify the client's needs, purpose of the website, target audience and business goals.

  • Design

    With all the information that has been gathered, i write a short script about website's purpose and goals and required functionalities then i create a prototype design using "Adobe XD", i pick the fonts, colors etc accroding to the type of website.

  • Front End Development

    Following the pre build prototype design, I write SEO friendly semantic HTML5 markup, I use Bootstrap to build beautiful, user friendly and cross browser compatible responsive layout means the website will render beautifully on any device and any browser, i use "npm", "Webpack" and "SASS" to customize Bootstrap and to write cleaner and moduler css/javascript code, i use "Git/Gihub" to esily maintain website's version and source code files and future updates and customizations.

  • Back End Development

    For dynamic user content driven websites i use "Django" and "Postgresql" to programme Database, an admin panel, user accounts, Interaction system between users, authentication and authorization system. Usually This type of websites has security concerns and i have deep understanding of the security holes and how to prevent them and Django is the most secure Back-End technology i know.

  • Testing

    After the completion of website development, i test it's functionality, device and browser compatibility, load speed and performance to insure that the website is bug free and ready for the lunch.

  • Lauch

    Once the green flag is shown, i launch the website to the server using FTP.

  • SEO

    SEO is about increasing your website's visibility in the organic search results of major search engines such as google. I have extensive background in SEO, basically everything in SEO revolves around keywords, page title, heading and content, i know how to put these together logically in the right place for all of my websites.
