About me

My name is Sagar Rai, I am a Freelace Web Developer. I have serious passion for web development and i enjoy learning new technologies and new concepts everyday. My goal is to build handsome websites for people so they can acheive their goals as well.

My workflow

Over the years i have built many websites in many different topics and experimented with many different techonologies, keeping in mind the main fundamentals of web development, one - websites should look unique and visually apealing, two - webpages should be bug free and should load fast, three - website should be responsive and should render beautifully on any device big or small. Here is how i build my websites from scratch which is based on my experience and years of experiments.

  • Gathering Information

    First of all i gather relavant information from the client so that i can identify the client's needs, purpose of the website, target audience and business goals.

  • Design

    With all the information that has been gathered, i write a short script, i plan the design strategies then i create a prototype layout using Adobe xd, pick the fonts, colors etc accroding to the type of website and target audience.

  • Front end development

    Accroding to the prototype design, I write SEO friendly semantic HTML5 markup, I use Bootstrap to build beautiful, user friendly and cross browser compatible responsive layout means the website will render beautifully on any device and any browser, i use npm and Webpack to customize Bootstrap using SASS and to write cleaner and moduler css/javascript code, i use Git/Gihub to esily mentain it's source codes and future updates and customizations.

  • Back end development

    For dynamic user content driven websites i use Django and Postgresql to programme Database, an admin panel, user accounts, Interaction sytem among users, authentication and authorization system. This type of website has security concerns and i have deep understanding of the security holes and how to prevent it and the Django is the most secure back end technology i know.

  • Testing

    After the completion of website development, i test it's functionality, device and browser compatibility, load speed and performance to insure that the website is ready for the lunch.

  • Lauch

    Once the green flag is shown, i launch the website to the server using FTP.

  • SEO

    SEO is about increasing your website's visibility in the organic search results of major search engines such as google. I have extensive background in SEO, basically everything in SEO revolves around keywords, page title, heading and content, i know how to put these together logically in the right place for all of my websites.
